## 1. 문제 정의

### 1.1 현재 상황 (As-Is)

처음 해외 수출을 시도하는 중소기업은 다음 문제에 직면합니다:

- **높은 비용**: 기존 컨설팅 업체는 FDA 인증에 2,400만원 수준 청구
- **긴 처리 기간**: 평균 60일 이상 소요
- **정보 비대칭**: 인증·허가가 무엇인지, 뭐부터 해야 하는지 모름
- **신뢰 부재**: 일반 컨설팅은 관세사 자격 없이 운영, 법적 효력 불명확

### 1.2 목표 상황 (To-Be)

- **합리적 비용**: FDA 160만원 (기존 대비 93% 절감) - AI 자동화로 비용 절감
- **빠른 처리**: AI 신청서 자동 작성 + 전문 관세사 검토로 평균 30일 내 처리
- **정보 제공**: 무료 라벨링 서비스로 인증 과정 체험
- **전문성 보장**: AI 초안 생성 + 11년 경력 전문 관세사 검토, 전문가배상책임보험

### 1.3 왜 지금인가? (Why Now)

- K-Food, K-Beauty 수출 붐으로 중소기업 해외 진출 수요 급증
- AI 기술 발전으로 문서 자동화가 가능해져 비용 절감 실현
- 기존 대형 컨설팅 업체의 높은 비용에 대한 시장 불만 존재

| 2   | 소형 컨설팅 B   | 식품 라벨링 서비스 | 저렴, 접근성   | 관세사 자격 無, 통관 미지원   |
| 3   | 관세사 사무소 C | 통관 서비스        | 전문성, 신뢰   | 인증·허가 미지원, 마케팅 부족 |

### 차별화 포인트

| 항목      | 경쟁사     | 우리               | 왜 더 나은가?                                   |
| --------- | ---------- | ------------------ | ----------------------------------------------- |
| 비용      | 2,400만원+ | 160만원~           | AI 신청서 자동화 + 전문 관세사 검토로 비용 절감 |
| 처리 기간 | 60일+      | 30일               | AI 자동 작성 + 관세사 검토로 시간 단축          |
| 원스톱    | 인증만     | 인증+통관          | 인증부터 통관까지 원스톱 지원                   |
| 보험      | 없음       | 전문가배상책임보험 | 실패 시 손해 보상                               |

### 킬러 질문 대응

> "대형 컨설팅이 같은 가격으로 서비스를 제공하면?"

- 대형 컨설팅은 구조적으로 고비용 체질 (대규모 인력, 사무실 유지비 등)
- 우리는 **AI 인증허가 신청서 자동화 + 전문 관세사 검토** 체제로 비용 절감
- AI가 반복 작업을 자동화하고, 전문 관세사가 최종 검토하여 품질 보장
- 전문가배상책임보험으로 안전망까지 제공

## 2. 사용자 페르소나

### 2.1 주요 페르소나

| 항목        | 내용                                                        |
| ----------- | ----------------------------------------------------------- |
| 이름/별칭   | "첫 수출 준비중인 김대표"                                   |
| 나이/직업   | 40대 중소 식품/화장품 제조업체 대표                         |
| 핵심 니즈   | 합리적 비용으로 미국/EU 수출 인증 획득                      |
| 불편 포인트 | 인증 절차가 복잡하고 비용이 불투명, 누구를 믿어야 할지 모름 |
| 사용 상황   | 해외 바이어로부터 FDA 인증 요청 받은 후 구글 검색           |

### 2.2 페르소나의 하루 (시나리오)

1. 해외 바이어로부터 "FDA 인증서 보내달라"는 메일 수신
2. "FDA 인증 비용" 구글 검색 → 대부분 업체가 2,000만원대
3. "FDA 인증 저렴" 검색 → 새봄컨설팅 발견
4. 랜딩 페이지에서 가격표 확인 (160만원) → "이게 진짜야?"
5. 무료 라벨링 서비스 체험 → AI가 실제로 라벨 초안 생성
6. "생각보다 괜찮네" → 무료 견적 받기 클릭
7. 카카오톡으로 상담 → 정식 의뢰 결정

## 3. 사용자 스토리

### FEAT-1: 랜딩 페이지 (신뢰 구축)

- [x] 방문자로서, 서비스가 무엇인지 3초 안에 파악할 수 있어야 한다. 왜냐하면 첫인상이 중요하기 때문.

- [x] 방문자로서, 관세사 자격/경력을 확인할 수 있어야 한다. 왜냐하면 전문성을 검증하고 싶기 때문.
- [x] 방문자로서, 실제 사례/실적을 볼 수 있어야 한다. 왜냐하면 "정말 됐나" 확인하고 싶기 때문.
- [x] 방문자로서, 쉽게 문의할 수 있어야 한다. 왜냐하면 복잡하면 이탈하기 때문.

### FEAT-2: 무료 라벨링 서비스 (Global Label AI)

- [x] 사용자로서, 로그인 없이 바로 사용할 수 있어야 한다. 왜냐하면 진입장벽을 낮춰야 하기 때문.
- [x] 사용자로서, 제품/원재료 정보를 입력하면 라벨 초안이 생성되어야 한다. 왜냐하면 이게 핵심 가치이기 때문.
- [x] 사용자로서, EU/미국 규정 차이를 자동 적용받아야 한다. 왜냐하면 규정을 직접 찾기 어렵기 때문.
- [x] 사용자로서, 알러지 성분이 자동 감지/강조되어야 한다. 왜냐하면 실수를 방지하기 때문.
- [x] 사용자로서, 결과를 복사/다운로드할 수 있어야 한다. 왜냐하면 활용할 수 있어야 하기 때문.
- [x] 사업자로서, 서비스 사용자 이메일을 수집할 수 있어야 한다. 왜냐하면 후속 마케팅을 위해.

### FEAT-3: 견적 요청 (v1)

- [x] 방문자로서, 카카오톡 또는 이메일로 쉽게 문의할 수 있어야 한다.

## 7. 해결 가설과 실험

### 7.1 가설

> 만약 **무료 라벨링 서비스**를 제공하면, **처음 수출하는 사업자**가 **인증 과정을 체험**하고 **정식 의뢰로 전환**할 것이다. 왜냐하면 **진입장벽을 낮추면 신뢰가 쌓이기** 때문이다.

### 7.2 실험 루프

| 단계 | 내용                                       |
| ---- | ------------------------------------------ |
| 실험 | 무료 라벨링 서비스 사용자 중 전환율 측정   |
| 관측 | 이메일 수집 수, 견적 문의 수, 정식 의뢰 수 |
| 학습 | 전환율 5% 이상이면 성공, 미만이면 CTA 개선 |
| 다음 | 전환율 낮으면 → 상담 예약 기능 추가 검토   |
